Overview:
    This project focuses on building a smart weapon detection system using the Faster R-CNN algorithm. The main objective is to automatically detect weapons like guns in images and videos with high accuracy. The system identifies objects by drawing bounding boxes and providing confidence scores, making threat detection easier and more reliable.

How the System Works:
    The system follows a modular approach that includes dataset upload, preprocessing, model training, and detection. It uses annotated datasets to learn and recognize weapons even under challenging conditions such as poor lighting and complex backgrounds. Once the model is trained, it can be saved and reused without retraining, improving efficiency and usability.

Video Detection Feature:
    A key feature of the system is its ability to analyze videos frame-by-frame. This enables near real-time detection, making it suitable for surveillance applications in places like airports, railway stations, and other public areas. The system highlights detected weapons in each frame, allowing quick identification and response.

Performance & Accuracy:
    The system’s performance is evaluated using metrics such as accuracy, precision, recall, and loss. These metrics help ensure that the model performs consistently and delivers reliable results with minimal errors.

Benefits & Impact:
    The project reduces manual effort by automating weapon detection and enhances overall security. It is also scalable, allowing integration with larger security systems and the ability to handle large datasets. Overall, it provides an efficient and practical solution for real-world safety applications.
